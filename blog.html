10. Sep 22 - Farming Simulator Equipment Efficiency Calculator

10. Sep 22 - Farming Simulator Equipment Efficiency Calculator Please input the numbers below as they are in the game.
Beware:
1. add decimal numbers with "." (e.g. "3.2")
2. only as long as all inputs have the same system (e.g. all inputs speed km/h), the efficiencies can be compared. Different systems lead to different numbers resulting in a different efficiency number.
3. if a given equipment is doing two things at a time, only fill in the equipments width and speed once. Example: Seeder that also Fertilize, insert "Seed" data and let "Fertilize1" data empty.
4. as of now the field related calculations must done with "km/h" and "meter" based inputs.

24. Dec 19 - Building this Website

I always wanted to have my own website, for experimenting with this 'Web Dev' stuff and also for presenting myself to the world. The target of this website is to be as simple as possible, among other sources, I was inspired by fefe and his rather simple web blog. I want to go one step further and only use HTML and CSS. This of course limits the visual 'wow'-effect but that's OK. After I got my domain, I needed a hoster. First I tried DigitalOcean, they are good and not free. I will keep them in mind for later projects but for my low requirements GitHub Pages, for free, seems good. The tutorial is very good, even though there is not much to do in the first place. Only the DNS is problematic as no nameserver IPs are listed, weird. My registrar had a support site specifically for GitHub where I could find these IPs.
