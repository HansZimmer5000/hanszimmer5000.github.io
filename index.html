24. Dec 19 - Setup this Website

I always wanted to have an own website, for experimenting with this 'Web Dev' stuff and also for presenting myself to the world. The target of this website was to be as simple as possible, among other sources, I was inspired by fefe and his rather simple web blog. I want to go one step further and only use HTML and CSS. This of course limits the visual 'wow'-effect but that's OK.

After I got my domain, I needed a hoster. First I tried DigitalOcean, they are good, but not free. So after a pause on working on this website I moved to GitHub Pages, which is free.
The hardest part in moving to GitHub was the DNS which was solved after few minutes (everything else took only moments). My registrar had a support site specifically for GitHub where I could find the GitHub Pages nameserver IPs. There were not listed in the official tutorial from GitHub.

Right now it seems working smooth for my very low requirements. Let's see how this turns out in the future.
